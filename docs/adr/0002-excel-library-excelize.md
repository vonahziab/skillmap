# ADR-0002: Генерация Excel — excelize/v2

## Статус
Принято

## Контекст
Результат сбора должен сохраняться в `.xlsx` со стилизацией (заливка,
шрифты, границы, freeze panes, альбомная ориентация, merged cells).

## Решение
Используем `github.com/xuri/excelize/v2`.

## Обоснование
- Единственная зрелая чистая-Go библиотека для `.xlsx` с полным контролем
  над стилями ячеек, merge, freeze panes, ориентацией листа.
- Не требует cgo или внешних бинарников — сохраняет цель «один .exe без
  зависимостей» (ADR-0001).
- Активно поддерживается, широкое покрытие OOXML-функций.

## Последствия
- Единственная внешняя зависимость проекта — оправдана, так как реализовать
  формат `.xlsx` вручную нецелесообразно.
- Обновления библиотеки нужно отслеживать вручную (go.mod), но API стабилен
  между минорными версиями v2.
